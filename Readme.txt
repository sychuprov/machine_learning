ПРОЕКТ "ОТТОК КЛИЕНТОВ"

ПОСТАНОВКА ЗАДАЧИ:

Из «Бета-Банка» стали уходить клиенты. Каждый месяц. Немного, но заметно. Банковские маркетологи посчитали: сохранять текущих клиентов дешевле, чем привлекать новых.
Нужно спрогнозировать, уйдёт клиент из банка в ближайшее время или нет. Предоставлены исторические данные о поведении клиентов и расторжении договоров с банком.
Необходимо построить модель с предельно большим значением F1-меры, минимальный порог 0.59. 

МОДЕЛИ:

Для исследования были взяты три модели
1. RandomForestClassifier - случайный лес
2. DecisionTreeClassifier - дерево решений
3. LogisticRegression - логистическая регрессия
